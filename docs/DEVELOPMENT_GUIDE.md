# 开发指南

## 开发环境设置

### 必需组件
1. Visual Studio 2022 或更高版本
   - 安装 .NET 桌面开发工作负载
   - 安装 C# 开发工具
   - 安装 Git 工具

2. .NET 8.0 SDK
   - 下载并安装最新版本
   - 配置环境变量

3. Google Chrome 浏览器
   - 安装最新版本
   - 记录安装路径

### 推荐工具
- Visual Studio Code
- ReSharper/Rider
- Postman
- Fiddler
